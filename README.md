# AV MEDIA | Boutique Video Agency Portfolio

Gallery-first cinematic landing page built with Astro 5 and static-first performance constraints.

## Stack

- Astro 5 (static output)
- Tailwind CSS
- Zero-JS page architecture (no client hydration)

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

The build process runs media optimization first:

```bash
npm run media:optimize
```

## Architecture Notes

- No interactive islands or client-side framework hydration.
- Hero uses static HTML video layering for cinematic impact.
- Gallery uses static cards and CSS hover effects only.
- Mobile layout stacks cards vertically.

## Performance Strategy

- Deep-charcoal base color and immediate text render for low-latency FCP.
- Hero video uses `preload="none"` and poster fallback.
- AVIF/WebP image negotiation with SVG fallback.
- Inlined stylesheets via Astro build settings for this single-page project.
